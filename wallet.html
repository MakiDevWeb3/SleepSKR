Wallet

Optional wallet support

Wallet support is optional and does not unlock the app. If a user connects one, SleepSkr only reads the public address and the wallet remains in control.

What it can do

Public address only

  • Guest mode available Users can use the app without a wallet.
  • Public address display A connected wallet can show its public address locally.
  • Voluntary support Donations are optional and user-approved.

What it cannot do

Private keys stay private

  • No private key access SleepSkr never reads the seed phrase or private key.
  • No silent transfers Any donation needs explicit approval in the wallet app.
  • Irreversible blockchain Once broadcast, blockchain transfers cannot be undone.
